# Forest-Cover-Type-Prediction
🌲 Forest Cover Type Prediction — Classifies 7 forest cover types using cartographic data (elevation, soil, wilderness area) from 15,120 samples. Implements Decision Tree, Random Forest, KNN &amp; XGBoost. XGBoost achieves best accuracy. Includes EDA, feature importance analysis, and model comparison. Python Scikit-learn XGBoost
